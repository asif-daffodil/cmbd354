# CMBD354 — Full‑Stack Web Development

Course workspace organized by class days and homework submissions.

## Quick start

- Open any `index.html` (or other `.html`) file in your browser.
- Recommended (VS Code): install **Live Server** and use **“Open with Live Server”** for auto-reload.

## Repository structure

- Each **Day-#** folder contains classwork and/or homework for that date.
- Common patterns inside a day:
	- `cw/` = classwork
	- `hw/` = homework (often grouped by student name)
	- `css/`, `images/` = assets used by that page
